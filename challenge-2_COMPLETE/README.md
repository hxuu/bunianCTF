## BunianCTF 2nd Challenge

### Description

### How to run the code

```bash
# make sure you're inside this directory
docker build -t bunian:jail .
docker run -d -p 5000:5000 bunian:jail
```

> [!TIP]
> You can interact with the challenge using `nc localhost 5000`
